# Week 1 - Day 1 SQL Practice

## Overview
This folder contains my Day 1 SQL practice exercises focused on fundamental SQL concepts and query-writing techniques.  
The practice covers basic data retrieval, filtering, aggregation, sorting, joins, and nested queries using SQL.
## Topics Covered

### 1. Basic Queries
- SELECT statements
- Retrieving all columns and specific columns
- Filtering records using WHERE clause

### 2. String Queries
- Pattern matching using LIKE
- String filtering techniques
- Working with text-based conditions

### 3. Data Queries
- Conditional data retrieval
- Comparison operators
- Logical operators (AND, OR, NOT)

### 4. Aggregate Queries
- COUNT()
- SUM()
- AVG()
- MIN()
- MAX()

### 5. GROUP BY Queries
- Grouping records based on columns
- Applying aggregate functions on grouped data

### 6. HAVING Clause Queries
- Filtering grouped records
- Using conditions with aggregate functions

### 7. ORDER BY Queries
- Sorting records in ascending and descending order
- Multi-column sorting

### 8. Join Queries
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Combining data from multiple tables

### 9. Nested Queries
- Subqueries
- Using queries inside SELECT and WHERE clauses

--
## Learning Objectives
- Understand SQL query structure
- Improve database querying skills
- Practice data filtering and aggregation
- Learn relational database operations using joins and subqueries
