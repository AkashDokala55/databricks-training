# Week 2 SQL Practice

## Overview
This folder contains my Week 2 SQL practice exercises focused on intermediate and advanced SQL concepts.  
The practice includes nested queries, correlated subqueries, joins, aggregations, filtering techniques, and moderate-level SQL problem solving.

---

### 1. Nested Queries
- Subqueries in SELECT statements
- Subqueries in WHERE clauses
- Scalar and multi-row subqueries

### 2. Correlated Queries
- Correlated subqueries
- Department-wise comparisons
- Salary and age-based filtering

### 3. Aggregate Functions
- COUNT()
- SUM()
- AVG()
- MAX()
- MIN()

### 4. GROUP BY and HAVING
- Grouping records
- Filtering grouped results
- Department-wise calculations

### 5. Join Queries
- INNER JOIN
- LEFT JOIN
- Combining multiple tables

### 6. Advanced Filtering
- Highest salary queries
- Average-based comparisons
- Date-based filtering

### 7. Ordering and Ranking
- ORDER BY
- Highest and nth highest salary queries
- Sorting aggregated results
