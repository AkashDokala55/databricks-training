# Joins in MySQL

## Overview
This module covers different types of SQL joins used to combine data from multiple tables based on related columns.

The practice includes employee, department, and project-based relational queries using INNER JOIN, LEFT JOIN, RIGHT JOIN, SELF JOIN, and FULL OUTER JOIN concepts.

---

## Topics Covered

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN (Concept)
- SELF JOIN
- Employee–Manager Relationships
- Department Mapping
- Project Assignments
- NULL Handling in Joins
- Multi-table Queries

---

## Practice Areas

- Employee and manager relationships

- Employees with projects
- Departments without employees
- Employees without projects
- Aggregation with joins
- Real-world relationship mapping
